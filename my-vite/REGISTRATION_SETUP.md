# Налаштування форми реєстрації

## Опис

Форма реєстрації користувачів з інтеграцією Django REST API.

## Налаштування

### 1. Створіть файл `.env` в корені проекту `my-vite`:

```env
VITE_API_BASE_URL=http://localhost:4099/api
```

### 2. Запуск Django REST API сервера:

```bash
cd "Django REST API"
start_api.bat
```

Сервер буде доступний за адресою: `http://localhost:4099`

### 3. Запуск React додатку:

```bash
cd my-vite
npm install
npm run dev
```

Додаток буде доступний за адресою: `http://localhost:5173`

## Функціонал

### Валідація на клієнті:
- Логін: мінімум 3 символи, лише літери, цифри та підкреслення
- Email: коректний формат email
- Ім'я та прізвище: мінімум 2 символи, лише літери
- Телефон: валідний формат номера
- Пароль: мінімум 8 символів, велика літера, мала літера, цифра, спеціальний символ
- Зображення: JPG, JPEG, PNG, GIF, WEBP (макс. 5MB)

### Валідація на сервері:
- Повна валідація всіх полів
- Перевірка унікальності email та username
- Обробка та оптимізація зображень (3 розміри)

### API Endpoint:
- `POST /api/users/register/` - реєстрація користувача

### Відповідь API:
```json
{
  "refresh": "token",
  "access": "token"
}
```

## Використання

1. Заповніть форму реєстрації
2. Натисніть "Зареєструватися"
3. При успішній реєстрації ви побачите повідомлення про успіх
4. При помилках валідації ви побачите відповідні повідомлення
